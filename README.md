# Тестовое задание

Проект состоит из двух независимых частей:

1. Pixel Perfect верстка макета (HTML/CSS/JS)
2. Разработка jQuery-плагина

---

# Часть 1 — Верстка макета

## Описание

Необходимо реализовать pixel perfect, семантическую и отзывчивую верстку макета, используя:

- Чистый HTML
- Чистый CSS
- Нативный JavaScript
- Без использования фреймворков и сторонних библиотек

### Кастомные компоненты

Реализовать собственные UI-компоненты формы с валидацией без сторонних библиотек:

- Select
- Checkbox
- input

---

# Часть 2 — jQuery Plugin

## Описание

Необходимо разработать плагин для jQuery, который:

- Подключается к выбранному DOM-элементу
- Отслеживает положение курсора внутри элемента
- Измеряет время задержки курсора в текущей позиции
- Отправляет данные на переданный API с заданным интервалом

## Функциональные требования

### Инициализация

Плагин должен вызываться на элементе следующим образом:

```javascript
$('#element').cursorTracker({
  apiUrl: 'https://example.com/api'
  checkInterval = 30,
  sendInterval = 3000,
});
```

### Отслеживаемые данные

Плагин должен фиксировать:

- X координату курсора
- Y координату курсора
- Время задержки в текущей позиции
